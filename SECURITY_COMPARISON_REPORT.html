4. Detayli Once / Sonra Karsilastirmasi

C-1: super-worker JWT Dogrulamasi

ONCE (Guvenlik Acigi)
const authHeader = req.headers.get("Authorization"); if (!authHeader) { return ... "Yetkisiz erisim" } // Token DOGRULANMIYOR - herhangi bir deger kabul ediliyor
SONRA (Duzeltildi)
if (!authHeader?.startsWith("Bearer ")) { ... } const supabase = createClient(url, key, { global: { headers: { Authorization: authHeader } } }); const { data: { user }, error } = await supabase.auth.getUser(); if (authError || !user) { return 401; }

C-2: Kredi Karti Formu

ONCE (PCI-DSS Ihlali)
<input placeholder="1234 5678 9012 3456" /> <input placeholder="MM/YY" /> <input placeholder="CVV" /> // Ham kart bilgileri React state'inde // Hicbir tokenizasyon yok
SONRA (Guvenli)
// Ham input alanlari KALDIRILDI // Stripe guvenli odeme bilgilendirmesi: "Odeme islemi Stripe guvenli odeme altyapisi uzerinden gerceklestirilecektir." // VISA / MC / AMEX ikonlari (bilgi amacli)

H-1: API Anahtari URL'den Header'a Tasindi (3 Fonksiyon)

ONCE (Log Sizintisi Riski)
const url = `...generateContent?key=${API_KEY}`; fetch(url, { headers: { "Content-Type": "application/json" } });
SONRA (Guvenli)
const url = `...generateContent`; fetch(url, { headers: { "Content-Type": "application/json", "x-goog-api-key": API_KEY } });

H-2: CORS Kisitlamasi (4 Dosya)

ONCE (Herkes Erisebilir)
"Access-Control-Allow-Origin": "*" // Herhangi bir web sitesi istek yapabilir
SONRA (Origin Bazli)
const ALLOWED_ORIGINS = [ "https://fikoai.de", "https://www.fikoai.de", "https://fibu-de-2.vercel.app" ]; // Dinamik origin kontrolu

H-3: Admin Yetkilendirmesi

ONCE (Client-Side)
const SUPER_ADMIN_EMAIL = import.meta.env.VITE_SUPER_ADMIN_EMAIL; if (session.email === SUPER_ADMIN_EMAIL) { setUserRole("admin"); // TARAYICIDA! }
SONRA (Server-Side RLS)
// VITE_SUPER_ADMIN_EMAIL kaldirildi supabase.from("profiles") .select("role") .eq("id", session.user.id) .single() .then(({ data }) => { setUserRole(data.role); // RLS korumali });

M-2 + M-3: AI Hata/Yanit Sizintisi

ONCE (Bilgi Sizintisi)
error: "AI servis hatasi: " + errText // Tam Gemini API hata detayi istemciye raw: rawText // Ham AI yaniti istemciye gonderiliyor
SONRA (Filtrelenmis)
error: `AI servis hatasi: ${status}` // Yalnizca HTTP status kodu console.error(errText.substring(0, 300)) // Detay sunucu loguna, istemciye degil

M-4: MFA Sifresi Gizleme

ONCE
{mfaSecret} // TOTP sifresi acik metin olarak gorunuyor
SONRA
{"••••••••••••••••••••••"} // Tikla-goster toggle ile gizli // "(gostermek icin tikla)" etiketi

M-5: Kullanilmayan Service Role Key

ONCE
const SUPABASE_URL = Deno.env.get("SUPABASE_URL")!; const SUPABASE_SERVICE_KEY = Deno.env.get("SUPABASE_SERVICE_ROLE_KEY")!; // RLS bypass riski - kullanilmiyor ama yuklu
SONRA
// SUPABASE_SERVICE_ROLE_KEY tamamen kaldirildi // Yalnizca GEMINI_API_KEY tutuluyor const GEMINI_API_KEY = Deno.env.get("GEMINI_API_KEY");

M-6: Giris Dogrulamasi (super-worker)

ONCE
const { fileBase64, fileType, ... } = await req.json(); // Boyut limiti yok, tip kontrolu yok // Kural sayisi siniri yok (prompt injection)
SONRA
if (bodyText.length > 10_000_000) return 413; if (!ALLOWED_TYPES.has(fileType)) return 400; if (learningRules.length > 200) return 400; if (settingsRules.length > 100) return 400; // Dosya tipi + boyut + kural siniri

L-2: Sifre Politikasi

if (regPassword.length < 6) // NIST standartlarinin altinda
SONRA
