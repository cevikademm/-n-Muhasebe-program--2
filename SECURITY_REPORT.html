4. Kritik Bulgular

KRITIK C-1: super-worker JWT Dogrulamasi Eksik
supabase/functions/super-worker/index.ts : satir 17-23

super-worker fonksiyonu Authorization header'inin yalnizca var olup olmadigini kontrol ediyor, ancak JWT token'ini dogrulamiyor. Herhangi bir Authorization: Bearer herhangi_deger header'i ile istek gonderilebilir. Bu, Gemini API'sinin yetkisiz kullanimina ve maliyet artisina yol acar.

Cozum: analyze-bank/index.ts satirn 82'deki gibi supabase.auth.getUser() cagirarak JWT dogrulamasi eklenmeli.

KRITIK C-2: Guvenli Olmayan Kredi Karti Formu
components/AuthScreen.tsx : satir 262-281

Odeme ekrani; kart numarasi, son kullanma tarihi ve CVV icin ham HTML <input> alanlari kullaniyor. PCI-DSS uyumlu tokenizasyon (Stripe Elements vb.) entegrasyonu yok. Kullanici gercek kart bilgilerini guvenli olmayan bir forma girebilir.

Cozum: Ham input alanlari kaldirilip Stripe Elements veya benzeri PCI-DSS uyumlu odeme saglayicisi entegre edilmeli.

5. Yuksek Oncelikli Bulgular

YUKSEK H-1: API Anahtari URL Parametresinde
supabase/functions/*/index.ts (super-worker:95, invoice-analyzer:68, analyze-bank:126)

Gemini API anahtari URL query parametresi olarak gonderiliyor (?key=${GEMINI_API_KEY}). Query parametreleri sunucu log'larinda, proxy log'larinda ve tarayici gecmisinde gorunur. API anahtari istek header'inda gonderilmelidir.

YUKSEK H-2: Wildcard CORS (Access-Control-Allow-Origin: *)
Tum Edge Functions + _shared/types.ts

Tum edge fonksiyonlari herhangi bir origin'den gelen isteklere izin veriyor. Kotu niyetli bir web sitesi, kullanicinin tarayicisindaki gecerli token'larla cross-origin istekler yapabilir.

Cozum: Origin kisitlamasi: https://fikoai.de ve staging domain'leri.

YUKSEK H-3: Client-Side Admin Yetkilendirmesi
App.tsx : satir 97-110

Super-admin durumu client-side'da e-posta karsilastirmasiyla belirleniyor (VITE_SUPER_ADMIN_EMAIL). Saldirgan client-side kodu degistirerek admin UI ozelliklerine erisebilir.

YUKSEK H-4: Edge Function'larda Rate Limiting Yok
Tum supabase/functions/*

Hicbir edge fonksiyonunda hiz sinirlamasi uygulanmamiistir. C-1 bulgusiuyla birlikte, saldirgan sinirsiz Gemini API cagrisi yapabilir.
